# Glossary

Key terms, abbreviations, and concepts used throughout this project.

---

## Vehicle & Competition

| Term | Definition |
|------|-----------|
| **FSAE** | Formula SAE — a collegiate engineering competition where student teams design, build, and race open-wheel formula cars |
| **Endurance** | The longest dynamic event (~22 km), testing reliability and efficiency |
| **Efficiency** | Points awarded based on energy consumed during endurance relative to competitors |
| **DSS** | Design Spec Sheet — competition-required document listing all vehicle specifications |
| **CT-16EV** | UConn's 2025 FSAE Electric car |
| **CT-17EV** | UConn's 2026 FSAE Electric car (in design) |
| **LVCU** | Low-Voltage Control Unit — the car's main controller |

## Battery & BMS

| Term | Definition |
|------|-----------|
| **SOC** | State of Charge (%) — how much energy remains in the battery (100% = full, 0% = empty) |
| **OCV** | Open-Circuit Voltage — terminal voltage with no current flowing |
| **BMS** | Battery Management System — monitors and limits battery operation for safety |
| **P45B** | Molicel P45B — the 4.5 Ah lithium-ion cell used in the 2025 pack |
| **P50B** | Molicel P50B — the 5.0 Ah cell planned for the 2026 pack |
| **110S4P** | Pack topology: 110 cells in Series, 4 in Parallel (440 total cells) |
| **SOC Taper** | Progressive current reduction as SOC drops below a threshold |
| **Discharge Limit** | Maximum allowed current (A) — depends on temperature and SOC |
| **Coulomb Counting** | Method of tracking SOC by integrating current over time |
| **I²R** | Resistive power loss (heat) — proportional to current squared |
| **Equivalent Circuit** | Battery model using ideal voltage source + internal resistance |
| **Voltt** | About:Energy's battery simulation platform used to model cell behavior |

## Powertrain & Motor

| Term | Definition |
|------|-----------|
| **PMSM** | Permanent Magnet Synchronous Motor — the type of electric motor used |
| **IQ / ID** | Quadrature / Direct axis current — motor control variables |
| **Field Weakening** | Technique to operate motor above base speed by reducing magnetic flux |
| **Back-EMF** | Voltage generated by the spinning motor — increases with RPM |
| **Regen** | Regenerative braking — using the motor as a generator to slow the car and recharge the battery |
| **Gear Ratio** | Ratio between motor and wheel speed (3.6363:1 = 40/11 teeth, motor spins 3.6363× faster than wheels) |
| **Drivetrain Efficiency** | Fraction of motor power that reaches the wheels (92%) |

## Dynamics & Track

| Term | Definition |
|------|-----------|
| **Quasi-Static** | Simulation method assuming steady-state at each track segment |
| **Curvature (κ)** | Inverse of turn radius (1/m) — higher = tighter corner |
| **Grade** | Rise/run — slope of the road (0.02 = 2% uphill) |
| **Cd** | Drag coefficient — dimensionless measure of aerodynamic resistance |
| **Crr** | Rolling resistance coefficient — dimensionless, typically 0.015 for FSAE tires |
| **Lateral Acceleration** | Centripetal acceleration in cornering (g) |
| **Grip Factor** | Multiplier on baseline lateral grip (1.0 = dry, < 1.0 = slippery) |
| **Point Mass** | Vehicle model that treats the car as a single point with mass — no tire slip, no suspension |

## Data & Tools

| Term | Definition |
|------|-----------|
| **AiM** | AiM Technologies — manufacturer of the data logging system |
| **Telemetry** | Data recorded during vehicle operation (speed, current, temperature, etc.) |
| **GPS LatAcc** | Lateral acceleration measured by GPS (used for curvature extraction) |
| **GPS Radius** | Turn radius reported by GPS (10000 = straight or unknown) |

## Simulation

| Term | Definition |
|------|-----------|
| **Segment** | A 5-meter slice of the track with constant curvature and grade |
| **SimState** | Snapshot of all simulation variables at a given instant |
| **ControlCommand** | Driver's output: throttle/coast/brake with intensity |
| **SimResult** | Complete output of one simulation run (DataFrame + summary) |
| **Lookahead** | The next 5 segments visible to the driver strategy for anticipatory decisions |
| **Pareto Frontier** | The set of solutions where you can't improve one metric without worsening another |
| **Parameter Sweep** | Running many simulations with one parameter varied to study its effect |
